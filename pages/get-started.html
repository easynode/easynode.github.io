Get Started

Creating a New EasyNode Project

Create a new app and run with babel-node 


    git clone https://github.com/easynode/easynode-template.git
    git checkout origin/helloworld
    sh init_project.sh
    //There, you input company's name, project's name, port, author
    npm install

Now lift the server 


    cd bin
    sh dev_start.sh
At this point, if you visit (http://localhost:8899/) you will see the default home page. Now, let's get EasyNode to do cool stuff.

Create a new app and run with node in the container

* Step 1: run the container ``` docker run -d hujb2000/easynode:6.2.0 /bin/bash The command will output the container id: ef35b98c9cece510b21d2aae6b1501424877d304408eb11b634de6b0f8f69455 ``` * Step 2: Copy the code of easynode-helloworld to your host's directory ``` docker cp ef35:/usr/src/easynode/examples/easynode-helloworld easynode-helloword ``` mount the easynode-helloworld directory to the container: ``` docker run -ti -v `pwd`/easynode-helloworld:/usr/src/app -w /usr/src/app/ hujb2000/easynode:6.2.0 /bin/bash ``` * Step 3: generate the Scaffolding ``` root@8fac0aa67200:/usr/src/app# sh init_project.sh current working directory is : /usr/src/app now it is: Thu Jun 16 04:24:57 UTC 2016 welcome to easnynode!
